## B) Volumes
### 1. 
```bash
docker volume create kn05-vol

docker run -d --name kn05-vol1 -v kn05-vol:/data nginx
docker run -d --name kn05-vol2 -v kn05-vol:/data nginx
```
### 2.
container 1
```bash
docker exec kn05-vol1 sh -c "echo 'Nachricht von Container 1!!!!!! (: Blääää' >> /data/shared.txt"
```
### 3.
container 2
```bash
docker exec kn05-vol2 cat /data/shared.txt
```
