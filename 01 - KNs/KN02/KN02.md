# KN02



## A) Dockerfile I
### Dokumentiertes Dockerfile
```dockerfile                         
FROM nginx                                 # Verwendet das offizielle nginx Image als Basis
COPY static-html-directory /var/www/html   # Kopiert die HTML-Datei ins Arbeitsverzeichnis
EXPOSE 	80	                               # Exponiert Port 80 f√ºr den Webserver
```
### Build
```bash
docker build -t luka0731/m347:kn02a .
```
### Container starten
```bash
docker run -d -p 8082:80 --name kn02a luka0731/m347:kn02a
```
### Push in privates Repository
```bash
docker push luka0731/m347:kn02a
```
